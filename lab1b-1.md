# Reflection Questions
•	- What’s the role of a firewall in managing services?

ANS: The role of a firewall in a managing service is to prevent any intruder or data breaches from entering acting as a gatekeeper.

•	- How did SSH access deepen your understanding of Linux as a server?

ANS: It provides a hands-on environment for me to manage my system remotely, helping me to practice more of the Linux tools and concept.

•	- Why is file compression important in server contexts?

ANS: It helps optimize storage space and a faster file transfer time.

•	- How does user privilege management help secure systems?

ANS: It limits the access of other unauthorized users preventing them from editing the files unnecessarily.
